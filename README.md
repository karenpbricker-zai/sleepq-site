## Design System

### Brand Colors Used
- Deep Navy: #0A1628
- Warm Orange/Amber: #F5A623
- White: #FFFFFF
- Professional Blue: #4A90D9
- Light Background: #F0F4F8

### Typography
- Font: Inter (imported from Google Fonts)
- Weights: 400, 500, 600, 700, 800
- Professional hierarchy with clear visual distinction

### Common Elements
- Phone mockups with 40px border radius, dark bezel effect
- Subtle shadows for depth (0 20px 50px+ rgba patterns)
- SleepQ watermarks on all designs
- Consistent padding and alignment
- Responsive hover effects on buttons
